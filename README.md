# Windmill Nights
![A screenshot of the theme in action, using some generic code](https://windmillarts.b-cdn.net/Misc/Github%20Assets/Code_zz3QFHDYY2.png)  

A blue-tinted dark theme for Visual Studio Code and similar IDEs.  
For the little guys who live in a blue world.

## Install
1. Go to [VS Marketplace](https://marketplace.visualstudio.com/items?itemName=WINDMILLARTS.windmill-nights)
2. Click on the "Install" button  

Alternatively you can download the [latest release](https://github.com/WINDMILL-ARTS/Windmill-Nights/releases/tag/public) from Github.
